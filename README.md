# java8study

すのとらさんとJava8の勉強をするためのリポジトリです。
プロジェクトは全て以下の前提で作られています。

* Eclipseプロジェクト
* JavaSE 8
* Mavenを利用
* Lombokを利用

## java8study
ちょっとしたお題を行う、サンドボックス的なプロジェクトです。

## java8study.util
応用問題を行う時に利用できる、汎用的なクラス群です。

## java8study.mastermind
数字あてゲームを作成する応用問題です。

## java8study.minesweeper
マインスイーパーを作成する応用問題です。
